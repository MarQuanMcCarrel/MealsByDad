# MealsByDad


Each Recipe need to contain certain components so that it will interact will our front end properly

# Entities
## User
- needs username 
- ability to log in
- ability to like recipes
## Recipe
### Needs to have
- Name
- Description
- Instructions
- Category
- Ingredients
- Likes

# Components

- Login
- Recipe card
- Search
- Display list
- Resources (links to other helpful pages)
- Our bio page
